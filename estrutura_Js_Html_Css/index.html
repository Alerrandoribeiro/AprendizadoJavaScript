Atalhos do VS Code

Atalhos gerais

Vamos ver alguns ver alguns atalhos legais do VS Code que vão deixar a nossa vida mais alegre, supimpa e feliz.

ctrl + espaço: chama o autocompletar de um contexto.

alt+z: Quebra automaticamente de linha

alt+setas cima/baixo: Move a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha que o curso está.

shift+alt+setas cima/baixo: Copia a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha que o curso está.

ctrl + ponto e virgula : comenta código css
